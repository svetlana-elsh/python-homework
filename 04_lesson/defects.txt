БАГ-РЕПОРТ: Неполная документация метода capitalize
===================================================

ID ДЕФЕКТА: 001

СТАТУС: Открыт

ОПИСАНИЕ:
Документация метода capitalize() не полностью описывает его поведение.
Не указано, что происходит с регистром остальных символов строки.

ТЕКУЩАЯ ДОКУМЕНТАЦИЯ:
"Принимает на вход текст, делает первую букву заглавной и возвращает этот же текст"
Пример: `capitilize("skypro") -> "Skypro"`

ПРОБЛЕМНЫЕ СЛУЧАИ:
1. Не описано поведение для строк со смешанным регистром
   Вход: "SkyPro" -> Выход: "Skypro" (становятся строчными)
   
2. Не описано поведение для строк в верхнем регистре  
   Вход: "SKYPRO" -> Выход: "Skypro" (становятся строчными)

3. Фраза "возвращает этот же текст" может вводить в заблуждение,
   так как регистр остальных символов изменяется.

ПРЕДЛОЖЕНИЕ ПО ИСПРАВЛЕНИЮ:
Уточнить документацию метода:
"Принимает на вход текст, делает первую букву заглавной, а остальные - строчными.
Примеры:
- capitalize('skypro') -> 'Skypro'  
- capitalize('SkyPro') -> 'Skypro'
- capitalize('SKYPRO') -> 'Skypro'"

ЛИБО:
Изменить реализацию метода чтобы он сохранял регистр остальных символов.
